# 外音取り込み君

マイクから取り込んだ音をそのままイヤホンなどに流すツールです。

ブラウザで試す: https://zk-phi.github.io/gaionkun/

## アプリ版

Electron で単体のアプリとして動かせるようにしたものです。

https://github.com/zk-phi/gaionkun/releases/

(自分の OS にあったものをダウンロードしてください)
